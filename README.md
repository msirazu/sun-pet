# 🐾 Sun Pet - Pet Adoption Platform

Sun Pet is a full-stack web application designed to connect pet lovers with pets waiting for a forever home. Users can browse pets, submit adoption requests, and manage their activity, while pet owners can easily manage their pet listings.

## 🚀 Live Demo

https://sun-pet.vercel.app

## ✨ Key Features

User Authentication: Simple signup/login using Email/Password and Google Social Login.

1. Pet Browsing: View all available pets with advanced search, species filtering, and sorting options.

2. Adoption Requests: Authenticated users can request to adopt their favorite pets.

3. Dashboard: A complete management system where owners can Add, Update, or Delete listings and manage adoption requests (Approve/Reject).

4. Responsive Design: Fully mobile-friendly and accessible on all screen sizes.

## 🛠 Tech Stack

Frontend: Next.js, Tailwind CSS, HeroUI

Backend: Node.js, Express.js

Database: MongoDB

Authentication: Better Auth, JWT
